# 医疗健康助手 - 用户使用指南

## 系统概述

医疗健康助手是一个基于AI的智能医疗咨询系统，旨在为用户提供便捷的健康咨询、症状自查和疾病信息查询服务。本指南将帮助您了解如何有效使用系统的各项功能。

## 快速开始

### 系统访问

1. **本地部署版**：在完成安装后，通过浏览器访问 `http://localhost:5173`
2. **在线版**：访问系统提供的在线网址（根据实际部署情况提供）

### 系统登录

目前系统提供匿名访问模式，无需注册登录即可使用所有功能。未来版本将增加用户账户系统，用于保存个人健康记录和咨询历史。

## 功能使用指南

### 1. AI对话咨询

AI对话咨询功能允许您通过自然语言与智能助手交流，获取健康相关问题的回答和建议。

#### 使用步骤：

1. 在首页点击"AI对话咨询"卡片或从顶部导航栏选择"AI对话"
2. 在聊天界面的输入框中输入您的健康问题，例如：
   - "我最近总是头痛，可能是什么原因？"
   - "感冒应该吃什么药？"
   - "如何改善睡眠质量？"
3. 点击"发送"按钮或按Enter键
4. 等待AI助手回复您的问题
5. 您可以继续提问，AI助手会基于对话历史提供上下文相关的回答

#### 注意事项：

- 尽量详细描述您的症状、持续时间和其他相关信息
- AI助手的回答仅供参考，不能替代专业医生的诊断
- 对于紧急健康问题，请立即就医或拨打急救电话

### 2. 症状自查分析

症状自查功能允许您选择自己正在经历的症状，系统会分析这些症状并提供可能的疾病匹配和建议。

#### 使用步骤：

1. 在首页点击"症状自查分析"卡片或从顶部导航栏选择"症状自查"
2. 在症状选择界面，您可以：
   - 通过搜索框搜索特定症状
   - 按身体部位筛选症状
   - 从症状列表中选择您正在经历的症状（可多选）
3. 选择症状后，您可以为每个症状设置严重程度
4. 点击"提交诊断"按钮
5. 查看系统提供的诊断结果，包括：
   - 可能的疾病匹配（按匹配度排序）
   - 每种疾病的简要说明
   - 建议采取的措施
   - 是否需要紧急医疗救助的提示

#### 紧急情况提示：

如果系统检测到您选择的症状可能代表紧急健康状况，将会显示醒目的红色警报，并提供紧急联系电话和初步应对建议。在这种情况下，请立即寻求医疗帮助。

### 3. 疾病信息查询

疾病信息功能允许您浏览和搜索各类疾病的详细信息。

#### 使用步骤：

1. 从顶部导航栏选择"疾病信息"
2. 在疾病列表页面，您可以：
   - 浏览热门疾病
   - 使用搜索框搜索特定疾病
   - 按身体部位或严重程度筛选疾病
3. 点击某个疾病的卡片或名称，进入疾病详情页面
4. 在详情页面查看：
   - 疾病的详细描述
   - 常见症状
   - 可能的治疗方法
   - 预防措施
   - 风险因素

#### 相关功能：

- 在疾病详情页面，您可以点击相关症状查看更多信息
- 系统会提供该疾病的严重程度评估和是否需要就医的建议

### 4. 紧急指南

紧急指南提供各类紧急健康状况的识别和应对建议。

#### 使用步骤：

1. 从顶部导航栏选择"紧急指南"
2. 浏览不同类别的紧急情况，如：
   - 心血管急症
   - 呼吸系统急症
   - 神经系统急症
   - 创伤和外伤
   - 中毒和过敏反应
3. 点击特定类别的紧急情况，查看：
   - 关键症状和体征
   - 紧急应对措施
   - 何时需要立即就医
   - 预防提示

#### 重要提示：

紧急指南仅供参考，在实际紧急情况下，请立即拨打当地急救电话（如中国为120，美国为911等）寻求专业医疗救助。

### 5. 健康知识查询

健康知识功能允许您搜索各类健康相关知识和信息。

#### 使用步骤：

1. 从顶部导航栏选择"健康知识"
2. 在搜索框中输入您感兴趣的健康话题，如：
   - "糖尿病饮食注意事项"
   - "如何降低血压"
   - "儿童疫苗接种时间表"
3. 点击搜索按钮
4. 浏览搜索结果，查看相关的健康知识文章

## 系统提示和帮助

### 常见问题解答

#### Q: 系统的诊断结果是否准确？
**A:** 系统的诊断结果基于症状匹配算法，仅供参考，不能替代专业医生的诊断。如有健康问题，请咨询专业医疗人员。

#### Q: 如何获得更准确的AI回答？
**A:** 提供详细的上下文信息，包括症状描述、持续时间、个人健康状况和已采取的措施等。

#### Q: 系统是否会保存我的健康信息？
**A:** 目前版本采用匿名访问模式，不保存个人健康信息。未来版本的用户账户功能将提供健康记录保存选项。

#### Q: 如何报告系统问题或提供建议？
**A:** 在页面底部点击"联系我们"链接，填写反馈表单，或发送电子邮件至support@medical-assistant.com。

### 使用技巧

1. **提供详细信息**：无论使用AI对话还是症状自查，提供越详细的信息，获得的建议越准确。
2. **定期更新**：系统会定期更新疾病和症状数据库，建议定期使用最新版本。
3. **结合多功能**：如果对症状自查结果有疑问，可以使用AI对话功能进一步咨询。
4. **关注紧急提示**：请认真对待系统显示的任何紧急情况提示。

## 隐私和安全

我们重视用户隐私保护，采取以下措施：

- 目前版本采用匿名访问，不收集个人身份信息
- 所有数据传输使用加密技术
- 严格遵守相关数据保护法规
- 不会将您的健康信息分享给第三方（除非法律要求）

## 系统限制

请了解本系统的以下限制：

- 不能替代专业医疗诊断和治疗
- 不适用于紧急医疗情况（请直接联系医疗机构）
- 可能无法识别罕见疾病或复杂症状组合
- 健康建议基于一般医学知识，可能不适合特定个体情况
- 依赖于准确的用户输入和系统数据库的完整性

## 免责声明

本系统提供的所有信息和建议仅供参考，不构成医疗诊断、治疗或处方。在做出任何医疗决定之前，请咨询合格的医疗专业人员。系统开发者不对因使用本系统而导致的任何后果承担责任。

如遇紧急健康状况，请立即就医或拨打当地急救电话。
